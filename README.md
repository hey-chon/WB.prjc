# WB.prjc - Personal Website

A clean and modern personal website built as a training project. Showcasing my journey in web development.

## ✨ Features

- **Fully Responsive Design** – Works perfectly on mobile, tablet, and desktop
- **Interactive Hamburger Menu** – Smooth mobile navigation
- **Smooth Scrolling** – Clickable navigation with smooth scroll effect
- **Hover Effects** – Modern underline animation on navigation links
- **Social Media Links** – Direct links to Facebook, Instagram, and GitHub
- **Clean & Minimalist UI** – Simple and elegant design

## 🛠 Tech-Stack 

- **HTML5**
- **CSS3** (Flexbox, Grid, Media Queries)
- **JavaScript** (for mobile menu and smooth scrolling)
- **Font Awesome** (icons)

## 🚀 About the Project

This website was created by **Jimson Ilog**, a Grade 11 ICT student at Cavite Community Academy Inc.
